# 灵感记录技能

快速捕捉你的想法、创意、会议笔记和学习心得，自动归类存储到记忆库。

## 触发条件

当用户表达以下意图时触发：

- "帮我记录一个想法：..."
- "记一下：..."
- "/remember ..."
- "灵感记录：..."
- "记个笔记：..."
- "把这个存起来：..."
- "保存这个创意：..."

## 使用方法

### 方式 1：自然语言（推荐）

直接在对话中说：

> 帮我记录一个想法：算力平权是未来趋势，数据才是核心竞争力
> 
> 标签：#AI #战略 #产品创意

### 方式 2：命令格式

```
/remember 内容 --tags 标签 1 标签 2 --type 类型
```

### 方式 3：脚本调用

```bash
python scripts/record_inspiration.py "想法内容" --tags AI 战略 --type 想法
```

## 记忆类型

| 类型 | Emoji | 适用场景 |
|------|-------|----------|
| 想法 | 💡 | 产品创意、技术方案、商业洞察 |
| 会议 | 📋 | 会议纪要、决策要点、行动计划 |
| 笔记 | 📝 | 学习笔记、阅读心得、课程记录 |
| 目标 | 🎯 | 短期目标、长期规划、OKR |
| 数据 | 📊 | 重要数据、统计信息、调研结果 |
| 感悟 | 💭 | 生活感悟、思考、反思 |

## 工作流程

1. **接收输入** → 用户发送记录请求
2. **解析内容** → 提取内容、标签、类型
3. **智能关联召回** → 搜索历史灵感，查找相关内容（可选）
4. **自动研究判断** → 检测是否需要触发深度研究（新增）
5. **生成文件** → 按模板创建 `.md` 文件
6. **存储到 inbox** → `memory/inbox/` 目录
7. **确认反馈** → 返回存储位置 + 相关灵感（如有）+ 研究报告（如触发）

## 🧠 自动研究触发（新增）

### 触发规则

记录灵感后，**自动检测**内容是否包含以下特征，满足任一条件即触发 `inspiration-research`：

| 检测项 | 示例 | 触发 |
|--------|------|------|
| **包含问号** | "...是什么？"、"...吗？" | ✅ 触发 |
| **探索性词汇** | "为什么"、"如何"、"怎么"、"是否" | ✅ 触发 |
| **假设性表述** | "如果...那么..."、"应该...还是..." | ✅ 触发 |
| **对比选择** | "...还是..."、"...和...哪个..." | ✅ 触发 |
| **明确研究指令** | "研究一下"、"分析一下"、"深入探讨" | ✅ 触发 |

### 不触发场景

| 场景 | 示例 | 结果 |
|------|------|------|
| 简单陈述 | "算力平权是趋势" | ❌ 不触发 |
| 会议纪要 | "今天开会决定..." | ❌ 不触发 |
| 待办事项 | "记得明天提交报告" | ❌ 不触发 |
| 事实记录 | "今天学习了 OpenClaw" | ❌ 不触发 |
| 明确禁用 | "只记录，不研究" | ❌ 不触发 |

### 输出示例

**用户**：帮我记录一个想法：如果算力和知识都平权了，企业的核心竞争力是什么？

**输出**：
```
✅ 灵感已记录：memory/inbox/💡_2026-02-28-001.md

🔍 检测到探索性问题，自动触发深度研究...

📊 研究报告生成中：
1. 拆解核心问题
2. 搜索相关资料（5-8 篇）
3. 生成研究报告
4. 飞书发送文档

预计耗时：2-3 分钟
```

### 禁用自动研究

如不需要自动研究，可添加：
- "只记录，不研究"
- "先记下来"
- "--no-research"

## 🧠 智能关联召回

### 工作原理

当记录新灵感时，系统自动：

1. **语义搜索** → 使用向量相似度搜索 `memory/` 目录
2. **标签匹配** → 查找相同/相似标签的历史记录
3. **相关性评分** → 计算相似度分数（0-1）
4. **阈值过滤** → 仅展示相似度 > 0.35 的结果
5. **输出提醒** → 展示最多 3 条相关灵感

### 相关性评分公式

```
Score = α × Keyword_Score + β × Vector_Similarity + γ × Recency_Factor

其中:
- Keyword_Score = 关键词匹配权重 (0-1)
- Vector_Similarity = 余弦相似度 (0-1)
- Recency_Factor = 时间衰减因子 (0.8^天数差)
- α = 0.4, β = 0.5, γ = 0.1
```

### 输出示例

```
✅ 灵感已记录：memory/inbox/💡_2026-02-26-0015.md

🔗 发现相关灵感:

1. **算力平权与数据优先** (相关度：78%)
   路径：`inbox/💡_2026-02-26-0015.md`
   摘要：算力平权是未来趋势，数据才是企业的核心竞争力...

2. **知识平权讨论** (相关度：52%)
   路径：`inbox/💡_2026-02-10-1430.md`
   摘要：极空间 CEO 提到知识获取门槛正在降低...
```

### 禁用关联召回

如不需要自动关联，可添加 `--no-related` 参数：

```bash
python record_inspiration.py "内容" --no-related
```

## 每周整理

**时间**: 每周日 20:00（可配置）

**整理内容**:
- 📊 本周概览（记录数量、类型分布）
- 💡 想法归类（按主题分组）
- 📋 会议要点（提取决策和 action items）
- 🔗 关联发现（发现记忆之间的联系）
- ✅ 行动项追踪（检查完成情况）

## 最佳实践

### ✅ 推荐做法

- **及时记录** → 想到就记，不要等
- **添加标签** → 方便后续检索（2-5 个标签）
- **简要标题** → 用一句话概括核心
- **标注来源** → 记录触发场景（会议/聊天/阅读）

### ❌ 避免做法

- 不要写太长（>1000 字建议拆分）
- 不要不加标签（后续难检索）
- 不要混用类型（保持分类清晰）
